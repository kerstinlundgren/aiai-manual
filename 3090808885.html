Nya Aiai manual : Överblick [brukare]

Created by Kerstin Lundgren on Jul 11, 2025

I brukaröverblicken kan du som användare få en översikt av den information som finns sparad för respektive brukare, samt, beroende på rättigheter, hantera denna information. Olika användarroller ger olika tillgång till att se och ändra information.

Profilbild

Om ingen profilbild är inlagd för brukaren visas dennes initialer i kvadraten avsedd för detta. Genom att klicka på pennan i kvadraten kan du lägga till en profilbild för brukaren. Profilbilden kommer att visas på olika ställen där detta är adekvat, så som på arbetspass för medarbetare.

Personuppgifter

Genom att klicka på pennan i personuppgiftskortet kan du som användare se brukarens personnummer samt lägga till, ta bort och ändra personuppgifter för brukaren.

Detaljkort

Under brukarens personuppgiftskort visas ett antal detaljkort för att uppmärksamma dig som användare på eventuella åtgärder som behöver vidtas för brukaren. Detta kan t.ex. vara bemanningsbehov som saknas, beslut som håller på att gå ut eller rapportering som ska hanteras. Genom att klicka på detaljkorten dirigeras du vidare till adekvat vy för hantering.

Informationen i detaljkorten aggregeras i förekommande fall till widgetar på startsidan, där t.ex. info om alla brukare som saknar beslut samlas, för att användare med behörighet snabbt och enkelt ska få en överblick över de åtgärder som behöver göras.

Kompletterande information – brukare

I brukarens profil kan du som har rätt behörighet dokumentera kompletterande information, till exempel kontaktpersoner, personliga uppgifter och en beskrivning i personakten.

Kontaktpersoner

Under sektionen Kontaktpersoner kan du lägga till en eller flera personer som bör kontaktas vid behov – till exempel närstående eller gode män.

För varje kontaktperson kan du ange:

  • Namn (obligatoriskt)

  • Roll/relation (t.ex. “förälder”, “kontaktperson”)

  • Telefonnummer

  • E-postadress

  • Övrigt (fritextfält, exempelvis hur eller när personen bör kontaktas)

Så här gör du:

  1. Gå till brukarens profil.

  2. I sektionen Kontaktpersoner, klicka på Lägg till.

  3. Fyll i uppgifterna.

  4. Klicka på Spara.

För att redigera eller ta bort en kontaktperson, klicka på pennikonen bredvid kontaktpersonens namn.

Personlig information

Det finns ett fält för personlig information där du kan dokumentera viktig information.

Så här gör du:

  1. Gå till brukarens profil.

  2. I sektionen Personlig information, klicka på Redigera.

  3. Fyll i eller ändra informationen.

  4. Klicka på Spara.

Endast administratörer och medarbetsledare kan se denna information.

Beskrivning (i personakt)

Fältet Beskrivning visas i brukarens personakt. Här kan du lägga till en övergripande beskrivning om brukaren som är adekvat i personakten.

Denna information visas även i appen för medarbetare som har rätt att se brukarens personakt.

Så här gör du:

  1. Gå till brukarens personakt.

  2. Klicka på Beskrivning.

  3. Skriv in texten.

  4. Klicka på Spara.

📅 Blev kund

Fältet Blev kund anger datum då brukaren blev kund hos er. Denna uppgift kan även ha migrerats från Aiai Senior.

Så här gör du:

  1. Gå till brukarens profil.

  2. I sektionen Om (samma sektion som Personuppgifter), klicka på penn-ikonen.

  3. Ange eller ändra datum.

  4. Klicka på Spara.
