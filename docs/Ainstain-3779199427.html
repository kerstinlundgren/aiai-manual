AI Överblick [medarbetare]

I överblicken för medarbetare kan du som användare få en översikt av den information som finns sparad för respektive medarbetare, samt, beroende på rättigheter, hantera denna information. Olika användarroller ger olika tillgång till att se och ändra information.

Profilbild

Om ingen profilbild är inlagd för medarbetaren visas dennes initialer i kvadraten avsedd för detta. Genom att klicka på pennan i kvadraten kan du lägga till en profilbild för medarbetaren. Profilbilden kommer att visas på olika ställen där detta är adekvat, så som i schemavyer och chatt.

Personuppgifter

Genom att klicka på pennan i personuppgiftskortet kan du som administratör se medarbetarens personnummer samt lägga till, ta bort och ändra personuppgifter för denne.

Detaljkort

Över och under medarbetarens personuppgiftskort visas ett antal detaljkort för att uppmärksamma dig som användare på eventuella åtgärder som behöver vidtas för medarbetaren. Detta kan t.ex. vara pågående frånvaro, ledighetsansökningar eller rapportering som ska hanteras. Genom att klicka på detaljkorten dirigeras du vidare till adekvat vy för hantering.

Informationen i detaljkorten aggregeras i förekommande fall till widgetar på startsidan, där t.ex. info om alla medarbetares ledighetsansökningar samlas, för att användare med behörighet snabbt och enkelt ska få en överblick över de åtgärder som behöver göras.

Kompletterande information – medarbetare

Funktionen Kompletterande information gör det möjligt att lägga till ytterligare information om en medarbetare, utöver den som normalt anges under personuppgifter. Informationen är uppdelad i två delar: Personlig information och Kontaktpersoner. Denna information är endast synlig i webben.

Endast användare med administratörsbehörighet har möjlighet att lägga till, redigera och ta bort kompletterande information. Medarbetsledare kan läsa informationen, men inte redigera den.

Personlig information

Det finns även ett fritextfält för att spara personlig information som kan vara relevant i arbetsrelationen, till exempel önskemål, behov eller särskilda förhållanden.

Detta fält är endast synligt för administratörer och medarbetsledare, och kan exempelvis användas för anteckningar som inte hör hemma i andra delar av medarbetarens profil.

Så här gör du:

  1. Gå till medarbetarens profil

  2. I sektionen Personlig information, klicka på Redigera

  3. Skriv in informationen

  4. Klicka på Spara

Kontaktpersoner

Under sektionen Kontaktpersoner kan du lägga till en eller flera kontaktpersoner för medarbetaren, till exempel närstående som bör kontaktas vid behov.

För varje kontaktperson kan följande uppgifter anges:

  • Namn (obligatoriskt)

  • Relation/roll (fritextfält, t.ex. “partner”, “vän”, “förälder”)

  • Telefonnummer

  • E-postadress

  • Fritextfält – t.ex. särskild information om när eller hur personen bör kontaktas

Det är möjligt att lägga till flera kontaktpersoner för samma medarbetare.

Så här gör du:

  1. Gå till medarbetarens profil

  2. I sektionen Kontaktperson, klicka på Lägg till

  3. Fyll i de fält som är relevanta

  4. Klicka på Spara

För att redigera eller ta bort en kontaktperson, klicka på pennikonen vid personens namn.
