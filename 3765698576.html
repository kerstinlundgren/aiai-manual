Nya Aiai manual : Önska pass i schemat

Created by Kerstin Lundgren on Sep 11, 2025

Syfte

Funktionen Önska pass gör det möjligt för medarbetare att meddela att de kan arbeta ett obemannat pass, och för administratörer/medarbetsledare att skicka förfrågan till medarbetare om de kan bemanna ett pass. På så sätt underlättas dialogen kring bemanning och ger en smidigare hantering av obemannade pass.

Hur fungerar det?

För medarbetare

  • Förutsätter att administratören har aktiverat inställningen ”Medarbetare kan se obemannade pass” och att ”Obemannade pass kan önskas” är påslaget.

  • Du kan se obemannade pass i webb och app.

  • Du ser obemannade pass i enhets-/brukarschemat i webben samt i tabben “Obemannade” i Aiai Amigo.

  • På ett obemannat pass kan du ange om du kan arbeta passet på en skala från kan arbeta till kan inte arbeta

  • Administratören kan avisera dig om du blir tilldelad ett pass, du får då en notis.

    • I appen leder aviseringen direkt till det specifika arbetspasset.

    • I webben leder aviseringen till enhetsschemat där passet finns.

För administratörer och medarbetsledare (endast webb)

Aktivera funktionen

Funktionen går att aktivera per enhet, genom att i Schemainställningar → Visningsinställningar för enheten välja “Jag vill att mina medarbetare ska kunna önska arbetspass i schemat”

I Administration → Företagsinställningar → Schema kan huvudadministratören ställa in huruvida funktionen ska vara automatiskt aktiverad för nytillagda enheter eller inte.

Se önskningar/interaktioner i schemat och på pass

Alla pass som har minst en interaktion (önskan från medarbetare) visas med en “tumme”-ikon

För respektive obemannat pass kan du, genom att klicka på passet och därefter i fältet för att bemanna det, se vilka medarbetare som interagerat med passet. Dessa sorteras överst i listan av medarbetare.

Skicka förfrågan på pass

För obemannade pass kan du skicka en förfrågan till en, flera eller alla medarbetare i enheten huruvida de kan arbeta passet.

Bemanna pass

När du bemannar ett pass kan du välja “Avisera medarbetare” för att på så sätt skicka en notis till denne att han/hon fått passet. Notisen visas i såväl webben, som en pushnotis i appen.

Aviseringar

  • Medarbetare får avisering om:

    • Förfrågan på pass, när dessa skickas av medarbetsledare eller administratörer

    • De blivit tilldelade ett pass, enbart i de fall medarbetsledare/administratör väljer att skicka en avisering. Dessa går alltså inte ut automatiskt.

  • Administratörer/medarbetsledare får inga aviseringar kring önskningar på pass, då detta riskerar att ge väldigt många aviseringar. Istället visas dessa önskningar med en ikon på respektive pass.

Övrigt

  • Ändringar på passet tar inte bort gjorda önskningar.

  • Avpublicering av pass tar inte bort gjorda önskningar.

  • Önskningar från Aiai Senior migreras inte.

Steg för steg

Medarbetare: Önska pass

  1. Öppna schemat i appen eller webben.

  2. Leta upp ett obemannat pass (markerat i webbschemat, egen tabb i appen).

  3. Klicka på passet.

  4. Ange om du kan arbeta passet genom att klicka på önskad ikon.

Administratör/medarbetsledare: Hantera önskningar

  1. Gå till enhetsschemat i webben.

  2. Klicka på ett obemannat pass.

  3. Se vilka medarbetare som interagerat och när.

  4. Skicka eventuell förfrågan till en eller flera medarbetare.

  5. Tilldela passet till vald medarbetare och avisera denne om tilldelning.
