Nya Aiai manual : Se kollegors pass samt obemannade pass i webb och app

Created by Kerstin Lundgren on Sep 11, 2025

Syfte

Funktionen gör det möjligt för dig som medarbetare att i Amigo-appen se både obemannade pass och dina kollegors arbetspass/aktiviteter. Syftet är att skapa ökad överblick, samarbete och flexibilitet i planeringen.

Hur aktiverar jag funktionen?

Funktionen aktiveras av administratören på respektive enhet/brukarenhet.

  1. Gå till schema

  2. Välj “schemainställningar”

  3. Scrolla ned till “Visningsinställningar” och tillse att “Jag vill att mina medarbetare ska kunna se varandra i schemat” och “Jag vill att mina medarbetare ska kunna se obemannade pass i schemat” är ibockade.

Grundinställningar företag

Samma inställning finns under Administration → Företagsinställningar → Schema. Här kan huvudadministratören ange vilka inställningar som ska vara gällande för alla nyskapade enheter/brukarenheter.

Hur fungerar det?

Om visningsinställningarna anger att medarbetare ska kunna se varandras pass samt obemannade pass, kan detta göras i såväl webb som app.

I webben väljer medarbetaren den enhet eller brukare som han eller hon arbetar hos för att se passen.

I appens schemaflik finns det tre tabbar: Schema, Obemannade samt Kollegor.

Vad visas?

Obemannade pass

  • Du ser endast pass där alla aktiviteter ingår i en enhet du har access till.

  • Du kan klicka på ett obemannat pass för att se aktiviteter och tidtyper.

  • Om funktionen “Önska pass” är aktiverad kan du klicka på någon av ikonerna för att återkoppla hur väl passet fungerar för dig att arbeta.

  • Du kan kopiera passets datum och tid (för att till exempel dela i chatten).

Kollegors arbetspass

  • Visas endast om administratören har valt att tillåta det via inställningen ”Jag vill att mina medarbetare ska kunna se varandra i schemat”.

  • Du ser arbetspass för de enheter du har access till.

  • Du ser starttid, sluttid, enhet och vem som bemannar aktiviteten.

  • Systemet laddar automatiskt aktiviteter från dagens datum och 4 veckor framåt.

Filtrering

Användaren kan filtrera visningen på:

  • Enhet (om man har access till flera enheter)

  • Medarbetare/kollega
